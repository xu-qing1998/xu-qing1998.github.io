小程序上线

发表于 2020-12-22

1.打包发行

1.1在HbuilderX中选中项目，在工具栏中点击发行——>网站-H5手机版（仅适用于uni-app）
1.2复制文件

编译成功后，会在unpackage->dist中生成一个build文件夹，将其中的h5文件复制到你想要保存的位置

小程序上线就需要有服务器做中介，使得外网可以通过访问服务器加指定端口号来访问小程序。

常用服务器有Apache、Tomcat、IIS、Node，此处我们使用的是Apache服务器。这些服务器除了Node不需要安装软件外，其他服务器都需要安装对应软件。使用Apache服务器需要安装wamp。

2.安装wamp

WAMP是指在Windows服务器上使用Apache(服务器)、MySQL(数据库)和PHP(建站语言)的集成安装环境bai，可以快速安装配置Web服务器。

2.1安装wamp可能遇到的问题

安装wamp一定要在全英文路径下，如果是中文路径，有可能会导致服务器一直是橙色不变绿

2.1.1安装wampserver提示丢失MSVCR100.dll

出现这个错误需要我们安装C++环境，可以从https://www.microsoft.com/en-us/download/details.aspx?id=48145下载对应的c++环境安装（选择一个即可）：

​ Visual C++ Redistributable for Visual Studio 2010

​ Visual C++ Redistributable for Visual Studio 2012

​ Visual C++ Redistributable for Visual Studio 2015

将之前安装的wamp卸载，重装即可解决问题。

2.1.2wamp出现橙色

一般出现wampserver橙色主要的原因就是自己电脑的80端口被占用了，导致了wampserver无法再使用80端口了

（1）使用快捷键“WIN+R”打开运行，在运行中输入“services.msc”命令可以打开服务

（2）在右边的服务项目中找到 SQL Server Reporting Service或者MYSQL，右键查看该项目是否在运行中，如果运行就点击”停止”。

（3）重启服务器

3.Apache配置

3.1修改httpd.conf配置文件（安装路径\wamp\bin\apache\Apache2.4.4\conf\httpd.conf）

在编辑器中打开httpd.conf

​ （1）查找Listen,修改端口号（端口号要大于1024，1024以下端口被系统占用，不可以使用），如Listen 8888

​ （2）查找vhosts去掉Include conf/extra/httpd-vhosts.conf注释,Include conf/extra/httpd-vhosts.conf

去掉注释：

​ LoadModule proxy_module modules/mod_proxy.so

​ LoadModule proxy_http_module modules/mod_proxy_http.so
